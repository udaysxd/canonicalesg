# Licensing

Canonical ESG is designed as open semantic infrastructure.

To enable reuse, interoperability, academic research, regulatory modelling, and ecosystem development, Canonical ESG applies open licensing to all public artefacts.

---

## Specifications and Documentation

Canonical Disclosure Intents (CDI), Canonical Mapping Packs (CMP), and all human-readable documentation are licensed under:

**Creative Commons Attribution 4.0 International (CC BY 4.0)**

This permits:

- Copying and redistribution
- Adaptation and transformation
- Commercial and non-commercial use

Subject to attribution of Canonical ESG as the source.

---

## Machine-Readable Schemas and Code

JSON schemas, validation artefacts, and technical utilities are licensed under:

**MIT License**

This permits unrestricted use, modification, and integration within software systems.

---

## License Intent

The licensing model is designed to:

- Encourage widespread adoption
- Enable ecosystem tooling development
- Preserve attribution integrity
- Prevent proprietary capture of the core semantic model

Canonical ESG does not impose usage restrictions beyond attribution.
